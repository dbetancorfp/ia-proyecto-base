Consulta la API REST de Moodle del Campus Canario y muestra la respuesta formateada.

Función y parámetros: $ARGUMENTS

Ejecuta la llamada usando las credenciales del proyecto (token en `~/.config/cuaderno-aula/config.json`) y muestra:
1. La URL completa llamada (con token redactado)
2. La estructura JSON de la respuesta
3. Un análisis de los campos más relevantes para el Cuaderno de Aula

Base URL: https://www3.gobiernodecanarias.org/medusa/eforma/campus/webservice/rest/server.php
Token: leerlo de ~/.config/cuaderno-aula/config.json

# /develop-frontend

Implements the frontend plan generated by `/plan-frontend-ticket`. Works from Figma + plan, produces Vue 3 SFCs with PrimeVue and Playwright tests.

## Usage

```
/develop-frontend ai-specs/changes/GH-42_frontend.md
```

Or with a Figma URL:

```
/develop-frontend 42 https://figma.com/file/...
```

## What it does

1. Reads the plan file (and Figma design if provided)
2. Creates branch `feature/GH-42-frontend`
3. Implements in order: store → composable → components → page → tests
4. All components use `<script setup lang="ts">` — no Options API
5. Uses PrimeVue components — no custom UI for things PrimeVue covers
6. Writes Playwright E2E tests and Vitest component tests
7. Commits and opens a PR with `Closes #42`

## Figma integration

If you pass a Figma URL, Claude uses the Figma MCP to read the design directly and generate pixel-perfect components. Make sure the Figma MCP is configured in `.claude/settings.json`.
